# 🌐 Shahnaz Keya Portfolio Website

A modern, responsive, and animated personal portfolio website built using **HTML, CSS, and Font Awesome icons**. This project showcases professional identity, articles, activities, and personal information in a clean and elegant design.

---


---

## ✨ Features

- 💻 Fully responsive design (mobile + desktop)
- 🎨 Modern UI with smooth animations
- 🧭 Sticky navigation and animated menu hover effects
- 🖼️ Hero cover section with animated quote text
- 👩‍🏫 About section with personal introduction
- 📰 Articles grid layout
- 🏛️ Activities section with clean card design
- 🔗 Social media icon integration (Facebook, LinkedIn, WhatsApp)
- ⚡ Lightweight (no frameworks used)

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Font Awesome Icons
- Google Fonts (optional if added)
